## 使用ライブラリ
以下のOSSライブラリを使用しています。
- leaflet: opestreetmapを使用するためのjs/cssライブラリ
  - https://leafletjs.com/
- Leaflet.MovingMarker:  地図上のマーカーを移動させるためのLeafletプラグイン
  - https://github.com/ewoken/Leaflet.MovingMarker
